===ArrayList===

Array List is flexible, it is able to shift the size of an array and is able to remove and add an array in the easiest way.

How to create a Regular Array?
    • Method 1: Declare then initialize
        → int[] numbers = new int[5];    -> Creates array of size 5, all values are 0
        → String[] names = new String[3];    -> Creates array of size 3, all values are null

    • Method 2: Declare with values
        → int[] numbers = {10, 20, 30, 40, 50};
        → String[] names = {"Alice", "Bob", "Charlie"};

    • Method 1: Declare and initialize separately (NOT RECOMMENDED)
        → int[] numbers;
        → numbers = new int[]{10, 20, 30};


Syntax:
    • int first = numbers[0];    -> Get first element (index starts at 0)
    • int last = numbers[4];    -> Get element at index 4
    • .length;    -> Get array size (NOTE: no parentheses!)
    • numbers[0] = 100;    -> Change value at index 0
    • names[1] = "David";   -> Change value at index 1
    • .toString()   -> Prints out the array clean and neatly (Need to import Arrays)
