===ArrayList===

Array List is flexible, it is able to shift the size of an array and is able to remove and add an array in the easiest way.

How to create an ArrayList?
    → import java.util.ArrayList; // Don't forget to import

    → ArrayList<String> names = new ArrayList<>();   -> Empty list (no limit)
    → ArrayList<Integer> numbers = new ArrayList<>();   -> Use Integer, not int
    → ArrayList<String> list = new ArrayList<>(10);   -> Initial capacity (optional)

Syntax:
    .size() -> checks the how big(maximum values) the array is
    .add() -> adds a value inside an array
    .remove() -> removes a value inside an array
    .get() -> gets the value of an array by indicating its position(index)
    .set() -> changes the value of inside an array
